# Studvart Portfolio Site

Plain HTML, shared CSS and shared JavaScript.

Upload all files and folders to the root of the `Studvart.github.io` repository.

- Edit page content in the HTML files.
- Edit the visual theme once in `assets/css/styles.css`.
- Add small behaviours in `assets/js/script.js`.
- Place screenshots/profile images in `assets/images/`.
- Place CV or supporting documents in `assets/docs/`.
